# cut コマンドで ls コマンド出力の所有者名とフォルダ名を出力する
フォルダ名のみの抽出および、cutコマンドでは複数文字をセパレータにできないので、複数文字のまとめは sed コマンドにて実施
```sh
$ ls -l| sed -E -n -e 's/\s+/ /g' -e /^d/p | cut -f 3,9 -d ' '
```
